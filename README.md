# BlockDates - Your one stop crypto app



## Introduction

In this project, I, Bassey Goodluck, created a cryptocurrency app. We're going to use React and crypto APIs powered by https://rapidapi.com.

## INSTALL GUIDE

clone this repo by hitting `git clone https://github.com/Chrisszy123/electron-react-api.git` on your terminal

then `cd electron-react-api` 

run `npm install` this should install all the dependencies needed to run this project.

After the dependencies are done installing, 

run `npm start`. this would spin up the electron app in developement mode.

run `npm run build` to get the production build.
